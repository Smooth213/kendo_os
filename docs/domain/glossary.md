# Domain Glossary (ドメイン用語集)

エンジニアと競技運営者が共通認識を持つための言葉の定義です。

### 🤺 競技・試合用語
- **有効打突 (Valid Strike):** 審判によって認められた一本。システム上は `isIppon: true` として記録される。
- **反則 (Hansoku):** 競技規則違反。累積2回で相手に一本が与えられる。
- **代表戦 (Representative Match):** 団体戦で決着がつかない場合に行われる決定戦。

### ⚙️ システム・設計用語
- **決定論的リプレイ (Deterministic Replay):** 同一のイベント履歴から常に同一の結果を再現すること。
- **補償イベント (Compensating Event):** 過去を消すのではなく、「取り消し」という事実を積み重ねて修正すること。
- **権威端末 (Authoritative Device):** データの最終決定権を持つ、記録係の端末。
