# Core Philosophy (中核となる設計哲学)

Kendo_osは、単なるスコア表示アプリではなく、剣道という競技文化の正当性と歴史をデジタルの力で支えるためのプラットフォームです。
以下の5つの哲学は、システムのいかなる改修においても最優先され、決して損なわれてはならない不変の価値です。

## 1. Human Authority (人間の審判員の権威)
システムは審判員を置き換えるものではなく、その活動を支援し、情報の流れを円滑にするための裏方です。「有効打突」や「反則」の最終的な決定権は常に人間の審判員にあり、システムはそれを受け入れ、正しく記録する責務を負います。

## 2. Replay Truth (歴史的真実の保護)
一度行われた試合の結果は、その当時のルールにおいて「真実」です。将来ルールがアップデートされたとしても、過去のイベント履歴をリプレイした際に結果が変わる（歴史改変）ことは許されません。イベントストリームこそが唯一の不変の真実です。

## 3. Tournament Continuity (大会運営の継続性)
体育館という過酷な現場において「システムトラブルで大会が止まる」ことは最大のタブーです。ネットワークの切断、端末のフリーズ、同期の遅延といった状況下でも、目の前の試合を継続し、記録し続けるための回復力（Resilience）を追求します。

## 4. Determinism & Idempotency (決定論と冪等性)
同じイベント履歴と、同じルール設定が与えられれば、世界中のどの端末で何度計算しても、必ず寸分違わぬ同一のスコアと勝敗結果を出力します。この決定論性がシステムの信頼性を担保します。

## 5. No Silent Correction (勝手な自動補正の禁止)
システムが人間の意図を確認せずにデータを勝手に書き換えたり、自動で「正解」を導き出したりすることを禁じます。不整合が起きた場合は、必ずオペレーターの承認を経て、明示的な「訂正イベント」として記録を残します。

## 6. Knowledge Governance (第6の哲学)
「ドキュメントはコードと同様にテストされ、監査され、実行されるべき知識のランタイムである。取説の不在はシステムの欠陥と同義であり、知識の同期なきリリースはガバナンスに対する反逆である。」
