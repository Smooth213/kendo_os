# Governance Index (ガバナンス総合目次)

本ディレクトリは `kendo_os` の「人間主導・AI補助型」開発を支える、すべてのガバナンスと哲学の中枢です。（Phase 9 最終整理版）

## 🏛 1. Foundation (基礎哲学と不変条件)
- **Core Philosophy**: 競技運営システムとしての5つの絶対哲学。
- **Architecture Invariants**: AI・人間が守るべき技術的な絶対制約一覧。
- **Documentation Constitution**: ドキュメントの構造と管理責任。

## 🤖 2. Automation & AI (自動化とAI統治)
- **Governance Rules (YAML)**: AIが読み込むための機械可読なシステム不変条件。
- **AI Development Workflow**: AIにコードを書かせる際の安全な標準ワークフロー。
- **Automation Policy**: CIパイプラインによる哲学遵守の自動監視方針。

## 🛡 3. Process & Quality (プロセスと品質)
- **Review Checklist**: PR承認時に確認すべきガバナンス違反チェックリスト。
- **Chaos Validation Drill**: 現場のトラブルやカオスな状況を想定した実戦ドリル。

## ♻️ 4. Sustainability (永続性とライフサイクル)
- **Governance Lifecycle Policy**: バージョニング、Dead Rule の排除、長期アーカイブのルール。
- **Why-not Archive**: 過去の不採用案とその理由（歴史的教訓）。
