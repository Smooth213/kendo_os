# 🚨 緊急復旧ガイド（トラブル解決）

**システムが止まっても、大会は絶対に止めないでください。**
困ったときは、以下のYES/NO診断で行動を決めてください。

## 画面は動きますか？ (YES/NO 診断) {#recovery-flow}

- 【NO】画面が真っ暗、または固まって押せない 👉 すぐに「紙の記録用紙」に切り替えて試合を続けてください。（**👇 スクロールして下の「紙での運用」へ**）
- 【YES】画面は押せるが、通信エラーが出ている / 観客席の点数が変わらない 👉 そのまま「アプリで点数を入れ続けて」ください。（**👇 スクロールして下の「通信エラーの場合」へ**）

---

## 📝 紙での運用（タブレットが壊れた場合） {#recovery-paper}
**システムの停止は、大会の停止ではありません。**
1. 迷わず机上の「紙の記録用紙」と「赤鉛筆」を手に取ります。
2. 審判の判定を紙に書き、試合を最後まで終わらせます。
3. 後でタブレットが直った時に、紙を見ながらアプリに再入力すれば元通りになります。

## 📡 通信エラー・同期遅延（Wi-Fiが切れた場合） {#recovery-network}
画面に「オフライン」と出ても、**絶対にアプリを閉じないでください。**
手元のタブレットの中に、点数は安全に保存され続けています。
そのまま試合を続け、電波が直るのをお待ちください。

## ❌ スコアの押し間違い {#recovery-mistake}
間違えて点数を入れた場合は、画面の **【↩️ Undo（取り消し）】** ボタンを押してください。
何回間違えても、すぐに一つ前の状態に戻すことができます。
