# 試合画面の見方

現在行われている試合の状況を、リアルタイムで確認できます。

## 画面の見方 {#viewer-match-view}

![試合画面の見方](../../assets/manual_images/viewer_match_01_view.png)
① **残り時間**: 試合の残り時間が表示されます。
② **選手名とタスキの色**: 赤または白のタスキをつけた選手の名前が表示されます。
③ 現在の得点: 選手が取った「面」「小手」「胴」「突き」です。
④ **反則**: 反則の回数が表示されます。2回で相手に1本入ります。
⑤ **勝敗**: 試合が終わると、勝った選手の側に「勝」マークが表示されます。

## 表示が止まった場合 {#viewer-match-freeze}
体育館の電波状況により、数字の更新が遅れることがあります。
その場合は画面を閉じずに、そのままお待ちください。
電波が戻り次第、自動的に最新の点数に更新されます。
