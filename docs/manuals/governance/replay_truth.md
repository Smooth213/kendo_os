# 試合記録の正当性について

本文書は、Kendo Syncがなぜ「上書き」ではなく「記録の積み上げ（試合記録方式）」を採用しているかを定義します。

## 記録の不変性
本システムにおいて、一度発生した「一本」の記録は消去されません。
修正は「取り消し」という新しい記録として保存されます。

## 目的
- **透明性の確保**: 「誰がいつ操作したか」をすべて記録します。
- **記録の再現保証**: システムが止まっても、最初から記録を再生して復旧できます。
- **証跡の保護**: 改ざんが不可能な状態で記録を保護します。

## 運用ルール
- **誤操作の取り消し**: 間違えた場合は「取り消し」操作を行います。
- **事実の記録**: これにより、「一本」の事実と「修正」の事実を両方残せます。
- **主審優先 (Human Override)**: 自動判定ではなく、常に審判の判断が正解となります。

体育館という過酷な環境では、通信が不安定になることがあります。
しかし、本システムは全ての操作を端末内に保存し続けます。
たとえ通信が遮断されても、記録が失われることはありません。
