# 現場用クイックガイド（3分でわかる使い方）

**【記録係の皆様へ】** この用紙は、各コートの机に置いておくためのものです。  
**文字が大きく、最低限のことだけ**が書いてあります。

---

## 1. 最初の準備（管理者がやること）
1. **大会を作る**: 「大会新規作成」から名前とルールを決める。
2. **チームを入れる**: 出場する道場や選手を登録する。
3. **試合の箱を作る**: 進行表に合わせて試合枠を作成する。
4. **iPadを渡す**: 担当コートの画面を開いた状態で、記録係に渡す。

---

## 2. 試合の記録（記録係がやること）
主審の旗が上がったら、**赤か白のボタンを押すだけ**です。

- **ポイントを入れる**
  - 面（メ）・小手（コ）・胴（ド）・突き（ツ）・反則 をタップ。
- **間違えてしまったら？**
  - 慌てなくて大丈夫です。**「↩️ 取り消し（Undo）」ボタン**を押せば元に戻ります。
- **試合を終わらせる**
  - 勝負が決まったら、一番下の **「確定」ボタンを【長押し】** してください。

---

## 🚨 緊急時の対応（必ずお読みください）

体育館は電波が悪く、突然画面が動かなくなることがあります。

### Q. 「ネットワークが切断されました」と出た！
👉 **【そのまま入力を続けてください】** データはiPadの中に安全に保存されています。電波が戻れば、自動的に本部に送られます。絶対に画面を閉じたり、更新（リロード）しないでください。

### Q. iPadが完全に壊れた！電源が切れた！
👉 **【すぐに紙の記録に切り替えてください】** 1. 大会を止めず、手書きの記録用紙で試合を続行してください。
2. 試合終了後、別のiPadで「新しく試合を作り直し」、**紙の記録を見ながら同じ点数を入れて「確定」**させてください（これを**リプレイ**と呼びます）。
3. これで本部には正しい結果が記録されます。
