# ⏱️ 記録係 クイックガイド（机上用）

**システムが止まっても、絶対に試合を止めないでください。**

## 1. 試合の進め方
- **開始**: 選手が入場し、主審が「始め」と言ったら **【試合開始】** を押す。
- **点数**: 旗が上がったら、取った側の **【面】【小手】【胴】【突き】** を押す。
- **終了**: 「やめ」でタイマーの数字を押し、画面下の **【確定】** を **長押し** する。

## 2. 間違えたら？（取り消し操作）
- 押し間違えた時は、あわてずに **【↩️ Undo（取り消し）】** を押す。
- 何回でも一つ前に戻すことができます。

## 3. トラブルが起きたら？（緊急対応）
- 📡 **電波が切れた・オフラインと出た**
  👉 **そのまま点数を入れ続ける**（後で自動で直ります）。
- 🥶 **画面が固まった・押せなくなった**
  👉 **すぐに机上の「紙と赤鉛筆」で記録を続ける**（試合は止めない）。
