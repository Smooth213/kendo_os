# 📱 観客用スコア確認ガイド

お手持ちのスマートフォンで、現在の試合状況やスコアを確認できます。
**アプリのインストールやログインは不要です。**

## 1. アクセス方法（QRコード）
スマートフォンのカメラを開き、以下のQRコードを読み取ってください。
![QRコード](../../assets/manual_images/qr_viewer.png)
*(URL: https://kendo-sync.web.app/)*

## 2. 画面の見方
- **残り時間**: 画面の一番上に表示されます。
- **得点**: 「面(メ)」「小手(コ)」「胴(ド)」「突き(ツ)」が表示されます。
- **反則**: 2回で相手に1本入ります。
- **勝敗**: 勝った選手の側に「勝」マークが表示されます。

## ⚠️ ご注意
体育館の電波状況により、点数の更新が数秒遅れることがあります。
画面を閉じずに、そのままお待ちいただくと最新の点数に直ります。
