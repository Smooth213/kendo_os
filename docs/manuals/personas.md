# ユーザーペルソナ定義 (User Personas)

本ドキュメントは、マニュアルを執筆する際に「誰に向けて書いているか」の解像度を完全に固定するための定義である。書き手は、対象ペルソナのITレベルを超える用語を絶対に使用してはならない。

| Persona (対象者) | ITレベル | 現場での心理・状態 | ドキュメントの書き方 |
| :--- | :--- | :--- | :--- |
| **観客 (Viewer)** | 超初心者 | 我が子の試合進行やスコアを早く知りたい。 | IT用語を一切使わず、「見るだけ」で直感的に理解できるようにする。専門用語の解説も省く。 |
| **審判 (Referee)** | 低 | 試合の判定に集中したい。スマホ操作は苦手。 | システムの仕組みは説明せず、「システムより審判の判定が絶対である」ことのみを平易に伝える。 |
| **記録係 (Operator)** | 中 | 高校生〜一般。ミスを恐れて緊張している。 | 失敗しても直せる（取り消し可能）ことを強調し、図解と「次に押すボタン」だけを的確に示す。 |
| **大会責任者 (Admin)** | 中〜高 | 大会全体を管理し、トラブル時は復旧の判断を下す。 | 原因と結果のロジックを説明し、大会継続のためのYES/NO分岐を明確に提示する。 |
