# Documentation Navigation Tree (閲覧導線ツリー)

本ドキュメントは、Kendo Syncの「観客・閲覧系（Viewer）」と「運営・記録系（Operate）」のマニュアル導線を厳格に分離し、将来のアプリ内連携およびPDF出力の目次構造を固定するものである。

## Viewer (観客・閲覧系)
一般の観客や保護者がアクセスする導線。運営の裏側（操作方法やリカバリ手順）はここには含めない。

Viewer Home
 ├─ 試合画面
 ├─ 団体戦スコア
 ├─ 勝ち抜き戦スコア
 └─ FAQ

## Operate (運営・記録系)
大会運営者、記録係、審判向けの実務・緊急対応導線。

Operate Home
 ├─ 大会作成
 ├─ チーム登録
 ├─ 試合運営
 ├─ 成績管理
 ├─ 監査ログ
 ├─ 緊急復旧 (Recovery)
 └─ FAQ
