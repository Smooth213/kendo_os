# FAQ（よくある質問）

観客や保護者の方からよくある質問とその解決策です。

## 点数が変わらない {#faq-score-not-change}
**原因:** 体育館の電波状況が悪い状態です。点数が届いていない可能性があります。
**対処:** 画面を閉じずにお待ちください。電波が戻り次第、自動的に更新されます。

## 通信が切断された {#faq-network-disconnected}
**原因:** スマホの電波が圏外になっています。Wi-Fiが切断されている状態です。
**対処:** 電波の良いところへ移動します。大会の進行自体は止まっていません。

## 時間が止まった {#faq-time-stopped}
**原因:** 試合中に主審が「やめ」と宣告しました。時計が止まっている状態です。
**対処:** 試合が再開されれば再び動き出します。

## 画面が暗い / 選手名が違う {#faq-display-error}
**原因:** 前の試合結果のままになっています。運営側の入力が遅れているためです。
**対処:** 少し待つか、ブラウザの「更新」を押します。
