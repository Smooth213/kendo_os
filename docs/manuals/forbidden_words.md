# 禁止用語辞書 (Forbidden Words Dictionary)

対象者（特に観客、審判、記録係）向けのマニュアルにおいて、以下のシステム専門用語の使用を厳禁とする。

| NG (禁止用語) | OK (言い換え) | 理由 |
| :--- | :--- | :--- |
| **UI / Screen** | 画面（または操作画面） | 専門用語を避け、直感的に理解できる言葉にするため。 |
| **Replay** | 試合履歴の再構築 | 剣道におけるビデオ判定（インスタントリプレイ）と誤認されるリスクを防ぐため。 |
| **Projection** | 表示データ | 内部アーキテクチャの用語であり、一般ユーザーには意味不明なため。 |
| **Event Queue** | 同期待ち | システム内部のキュー構造は、一般ユーザーにとってノイズであるため。 |
| **Aggregate** | 試合データ | IT用語を排除し、日常語に置き換えるため。 |
| **Event Sourcing** | 試合記録方式 | 開発パラダイムの名称をマニュアルに露出させないため。 |
