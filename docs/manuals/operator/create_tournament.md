# 大会を作成する

新しく大会を開くための最初の準備です。
ここで設定したルールで、すべての試合が進行します。

## 画面の見方と手順 {#operate-create-steps}

!大会作成画面
① **大会の名前**: 「第1回 剣道大会」のように名前を入力します。
② **ルールの設定**: 試合時間や、勝敗の条件（2本先取など）を選びます。
③ **【保存】ボタン**: 最後にこのボタンを押すと、大会が作成されます。

## 間違えて作成した場合 {#operate-create-edit}
作成した大会は、後からでも名前やルールを修正（編集）できます。
一覧画面から対象の大会を選び、【⚙️ 設定】ボタンを押して直してください。
