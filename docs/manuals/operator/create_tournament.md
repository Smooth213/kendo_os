# 大会作成画面 (Create Tournament Screen)

## 概要
新しい大会（または錬成会）の基本情報を定義し、システム内に保存する。

## 使用タイミング
大会の準備段階、または練習試合を開始する前に管理者が使用する。

## 手順
1. 大会名、開催日、会場を入力する。
2. 競技形式（個人戦/団体戦/勝ち抜き戦）を選択する。
3. 「保存」をタップし、大会IDを発行する。

## 注意事項
- 大会名には、後で識別しやすいよう年度や回数を含めることを推奨。

## 障害時対応
- 入力中にアプリが終了した場合、データは保存されません。再度入力が必要です。

## FAQ
- **Q: 後から大会名を変更できますか？**
  - A: 管理者メニューの大会設定から変更可能です。
