# 成績管理と公式記録

全コートの試合結果が集約され、大会の最終的な記録を管理する画面です。

## 試合結果の確認 {#check-results}
① 画面にすべての試合の結果が一覧で表示されます。
② 勝敗や取得した本数に間違いがないか、本部席で最終確認を行います。

## PDF出力 {#export-pdf}
① 画面右上の【📄 PDF出力】ボタンを押します。
② 大会の公式記録として、印刷用のトーナメント表や成績表が保存されます。

## CSV出力 {#export-csv}
① 画面右上の【📊 CSV出力】ボタンを押します。
② 他の集計ソフト（Excelなど）で読み込める形式でデータが保存されます。

## 修正方法 {#fix-results}
公式記録に間違いを見つけた場合は、該当する試合をタップします。正しいスコアを上書き保存してください。
