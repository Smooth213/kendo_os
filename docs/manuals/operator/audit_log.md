# 監査ログ（操作履歴）の確認

「いつ、誰が、何を変更したか」をすべて確認できる画面です。
トラブルがあったときに、過去の操作をさかのぼって確認できます。

## 画面の見方 {#operate-audit-view}

!監査ログ画面
① **時間**: 操作が行われた時間が、新しい順に表示されます。
② **操作した人**: どの端末（第1コートなど）で操作されたかがわかります。
③ **変更内容**: 「赤に面を入れた」「取り消した」などの内容が表示されます。

## なぜすべて記録されるのか {#operate-audit-reason}
このシステムは、間違えた操作も隠さずに「すべて記録」します。
これにより、点数の経緯を正確に追跡できます。
