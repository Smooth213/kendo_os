# 試合記録画面 (Match Screen)

## 概要
剣道試合の有効打突（面・小手・胴・突き）および反則、タイマー操作をリアルタイムに記録する。

## 使用タイミング
試合開始から終了まで、コート記録係が常時使用する。

## 手順
1. 審判の「はじめ」に合わせて中央のタイマーをタップする。
2. 有効打突が発生した場合、該当する部位ボタンをタップする。
3. 2本先取または時間切れで勝敗が決まると「確定」ボタンが出現する。
4. 設定された操作（長押し等）で試合を確定させる。

## 注意事項
- 判定の間違いは、即座に「Undo（取り消し）」ボタンで修正すること。
- 確定後の修正には、管理者権限によるロック解除が必要。

## 障害時対応
- **ネットワーク切断時**: そのまま入力を継続してください。通信回復時に自動同期されます。
- **フリーズ時**: アプリを再起動し、試合一覧から当該試合を再開してください。状態は直前のイベントまで復元されます。

## FAQ
- **Q: 反則2回で1本になる処理は自動ですか？**
  - A: はい、全日本剣道連盟のルールに基づき自動加算されます。
