# 試合の記録（点数入力）

試合のタイマーと点数を記録する、最も重要な画面です。
**間違えてもすぐに直せるので、落ち着いて操作してください。**

## 画面の見方と操作順序 {#operate-match-steps}

![試合記録画面](../../assets/manual_images/operate_match_01_main.png)
① **【試合開始】ボタン**: 主審が「始め」と言ったら押します。タイマーが動きます。
② **点数ボタン**: 主審の旗が上がったら、取った選手側の【面】や【小手】を押します。
③ 【↩️ Undo】ボタン: 間違えた点数を取り消します。
④ **タイマー**: 主審が「やめ」と言ったら、ここを押して時間を止めます。

## 試合を終わらせるには {#operate-match-finish}

![試合確定操作](../../assets/manual_images/operate_match_02_finish.png)
① **【確定】ボタン**: 試合が終わったら、このボタンを **長押し** します。
長押しすると結果が保存され、自動的に次の試合に進みます。

## 通信エラーが出た場合 {#operate-match-offline}
体育館の電波が切れても、**絶対に画面を閉じないでください。**
そのまま点数を入れ続けてください。電波が戻った時に、すべて自動で送信されます。
